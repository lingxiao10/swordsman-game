## Features

### Combat System
- **4 Unlockable Weapons**: Start with your trusty cyber-saber, then unlock the Blaster, Cannon, and Mines as you level up
- **Combo Attacks**: Chain your attacks for maximum devastation
- **Dash Ability**: Dodge enemies with style (Shift key / Dash button)
- **Loyal Cyber-Dog**: Release your faithful companion to help fight enemies!

### Rescue Mechanics
- Save beauties scattered across the map to gain XP
- Level up to unlock new weapons
- Visit the **Safe House** to heal (get intimate with rescued beauties... for health reasons, of course)
- Each beauty can only heal you once — choose wisely!

### Boss Fight
- Survive until the timer runs out
- Face the **OMEGA ENTITY** in an epic showdown
- Victory awaits the worthy!

### Technical Goodies
- Stunning 3D graphics powered by **Three.js**
- Immersive audio with **Howler.js**
- Works on both **Desktop & Mobile**
- Virtual joystick for touch controls
- Bilingual support (English / 中文)

## Deployment

### It's ridiculously simple:

```bash
# 1. Clone or download this repository
git clone <your-repo-url>

# 2. Put it anywhere on a web server
#    (Apache, Nginx, Python SimpleHTTPServer, Node.js serve, etc.)

# 3. Open index.html in your browser

# That's it. You're a cyber-warrior now.
```

### Quick Local Server Options:

```bash
# Python 3
python -m http.server 8080

# Node.js (if you have 'serve' installed)
npx serve .

# PHP
php -S localhost:8080
```

Then visit `http://localhost:8080` and start slashing!

### No Build Required
This is a pre-built game. No `npm install`, no `webpack`, no tears. Just serve and play.
